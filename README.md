# Sistem Temu Balik Informasi (Information Retrieval)

Aplikasi pencarian sederhana menggunakan Python Flask.

## Fitur
- Algoritma: TF-IDF & Cosine Similarity
- Backend: Python Flask
- Frontend: HTML + Tailwind CSS

## Cara Menjalankan
1. Install library: `pip install -r requirements.txt`
2. Jalankan app: `python app.py`
